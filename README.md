A modern weather dashboard built with **Next.js**, **TypeScript**, **Tailwind CSS**, and the **Open-Meteo API**.

## Features

- Real-time weather data
- Responsive UI with Tailwind & DaisyUI
- Clean Next.js App Router structure
- Error monitoring with Sentry (optional)

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS + DaisyUI
- Open-Meteo API

## Getting Started

```bash
npm install
npm run dev
```
